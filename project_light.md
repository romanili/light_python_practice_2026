## Суть
Базовый вариант практики на Python без базы данных, без дубликатов, без резервных копий и без фреймворков. В нем остается только практичный минимум: обойти папку, отфильтровать файлы и собрать короткий отчет.

## Ограничения
- Использовать только стандартную библиотеку Python.
- Не подключать высокоуровневые библиотеки и фреймворки.
- Не брать готовые инструменты, которые делают обход, фильтрацию или отчет одной функцией.

## Этап 1. Запуск
**Задача:** принять путь к папке параметром запуска и проверить, что программа стартует.

**Критерий сдачи:**
- программа запускается;
- путь к папке передается в командной строке.

## Этап 2. Обход папки
**Задача:** рекурсивно перечислить файлы и папки, показать вложенность.

**Критерий сдачи:**
- обход работает;
- выводится список файлов.
- видна структура вложенных папок.

## Этап 3. Простой отчет
**Задача:** показать количество файлов, папок, общий размер и простой фильтр по расширению или имени.

**Критерий сдачи:**
- отчет формируется;
- данные читаемы и понятны.
- работает хотя бы один простой фильтр.

## Этап 4. Финал
**Задача:** добавить короткую справку по запуску, минимальную проверку входных данных.

**Критерий сдачи:**
- утилита завершает работу без ошибок на простом примере;
- есть короткая инструкция по запуску.
- есть журнал/отчет студента в `.txt`.

## Итог
Рабочий вариант: утилита принимает путь, обходит папку, фильтрует файлы, формирует короткий отчет и дает студенту шаблон журнала/отчета.
