# Консольный индексатор папок

Полный вариант учебной практики. Утилита обходит указанную папку, ведёт индекс
файлов в SQLite, ищет дубликаты по хэшу и сравнивает папку с резервной копией.

Используется только стандартная библиотека Python и модуль `sqlite3`.

## Требования

- Python 3.8+
- Внешние зависимости не нужны

## Запуск

```bash
python src/main.py <путь_к_папке>
```

Пример:

```bash
python src/main.py ~/Documents
```

При запуске создаётся база `data/app.db` (если её ещё нет).

## Этапы

1. **Каркас** — приём пути, структура проекта, схема SQLite. ✅ (текущий)
2. **Сканирование** — рекурсивный обход, сбор метаданных, сохранение индекса.
3. **Дубликаты** — хэширование файлов, поиск повторов.
4. **Резервная копия** — сравнение папки с бэкапом, отчёт о различиях.

## Структура

```
src/
  main.py           точка входа
  db.py             работа с SQLite (схема, инициализация)
  student_meta.md   метаинформация студента
data/
  app.db            база (создаётся автоматически, в git не хранится)
JOURNAL.md          журнал работ
```
